# Delivery Database

This code creates, populates and draws information from a database regarding online orders.

## Instructions for Build and Use

[Software Demo]()

Steps to build and/or run the software:

1. Run the entirety of the main.sql file to create and populate the database.
2. In the console use "pip install mysqlconnector" and "pip install matplotlib".
3. Run the program.py file to select what information to display from the database.

Instructions for using the software:

1. Upon running both files select an option for displaying data.

# Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* MySQL80
* Python 2026.2.0
